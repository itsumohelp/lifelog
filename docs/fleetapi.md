# Tesla Fleet API 実装ノウハウ

本ドキュメントは、Tesla Fleet API を用いた **参照系アプリケーション**の実装を通じて得られた知見・注意点・設計判断をまとめたものです。

本プロジェクトでは以下を前提とします。

- 個人向けアプリケーション
- 車両を **起こさない（wake up しない）**
- 欠測（取得できない日）が発生しうることを許容
- 日次スナップショットを主用途とする
